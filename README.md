## VitePress Theme 

Vitepress 知识付费主题。

本主题是基于VitePress默认主题做扩展，实现部分内容会员可见，做知识付费会员文档。

主题主页：[https://vitepress.mosong.cc](https://vitepress.mosong.cc)

>提示：    
>如果需要基于本主题功能继续定制化，建议Fork仓库之后，使用`git submodule add`方式添加到你的项目`docs/.vitepress/theme`目录后继续修改。


## 主要功能

- 微信扫码登录
- 微信扫码付费，开通会员后可阅读会员内容
